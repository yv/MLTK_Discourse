The Parts of MLTK_Discourse
===========================

The main part of MLTK_Discourse is the features_exml script, which calls
the main function in the module with the same name.

The ``--features`` flag allows to modify the set of linguistic features
that are extracted alongside the shallow and graph-based ones.
